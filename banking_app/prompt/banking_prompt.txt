Erstelle den vollständigen Quellcode für eine lauffähige prototypische Internet-Banking-Webanwendung. Die Anwendung soll Benutzerregistrierung, Login, Kontenübersicht, Transaktionshistorie und Überweisungen zwischen Konten ermöglichen. Verwende Python mit FastAPI für das Backend und SQLite als Datenbank. Der Code soll modular strukturiert sein und eine kurze Anleitung zum Starten der Anwendung enthalten.
